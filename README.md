# Proyecto_Python_JaimesAngeluxSuarezDaniel
Simulador de Gasto Diario es una aplicación de consola en Python que permite registrar, monitorear y analizar tus gastos en categorías como comida, transporte, entretenimiento, etc. Los datos se guardan en un archivo JSON, permitiendo hacer seguimiento a tus finanzas de manera sencilla y sin complicaciones.
